# Leave

The Leave module manages annual leave, sick leave, and other absence types.

## Requesting leave

1. Go to **Leave** → **New Request**
2. Select the leave type, dates, and reason
3. The system checks your balance — insufficient balance blocks submission
4. Submit for manager approval

## Leave types

Defined by HR Admin in Settings → Leave Types. Typical types: Annual Leave, Sick Leave, Emergency Leave, Unpaid Leave.

## Balances

Your leave balance shows entitlement, carried over days, adjustments, and used days. Balances are adjusted by HR Admin as needed.

## Team calendar

The team calendar shows everyone's approved leave so you can plan around colleagues' absences. The system warns if more than 2 people from the same department are on leave simultaneously.

## Approval workflow

Leave requests go to the manager for approval or rejection. You receive a notification when your request is actioned.
