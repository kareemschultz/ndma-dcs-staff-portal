# DCS Ops Center

Welcome to the internal documentation for **DCS Ops Center** — the operations management platform for NDMA Data Centre Services (DCS).

## What is DCS Ops Center?

DCS Ops Center replaces spreadsheets and WhatsApp coordination for the day-to-day running of NDMA's data centre. It covers every operational domain:

| Module | Purpose |
|--------|---------|
| **Work Management** | Task tracking, weekly updates, initiatives, and cycles |
| **Cycles** | Sprint-style planning periods grouping work items |
| **Incident Management** | Incident response, timelines, responders, and PIRs |
| **On-Call Roster** | Schedule management, swaps, history, and fairness reports |
| **Leave** | Leave requests, balances, and team calendar |
| **Temporary Changes** | Time-bounded change tracking with hard removal dates |
| **Access & Accounts** | VPN, Zabbix, AD, and all platform account governance |
| **Procurement** | Purchase requisitions, approvals, and vendor tracking |
| **Compliance** | Training records, PPE logs, and policy acknowledgements |
| **Appraisals** | Staff performance review records |
| **Contracts** | Employment and vendor contract expiry tracking |
| **Staff Directory** | Staff profiles, departments, and workload view |
| **Notifications** | In-app alerts for key system events |
| **Analytics** | Cross-module KPIs and operational health dashboard |
| **Audit Log** | Immutable record of every change in the system |
| **Import Pipeline** | Bulk-load staff and leave from spreadsheets |
| **Automation Rules** | Event-driven notifications and follow-ups |
| **Settings** | Departments, leave types, and system configuration |

## Quick start

1. Sign in at the DCS Ops Center app (email + password)
2. Your role determines what modules and actions you can access
3. All changes are permanently recorded in the Audit Log

## Roles

| Role | Access level |
|------|-------------|
| **Admin** | Full access — all modules, settings, user management |
| **HR / Admin Ops** | Staff, leave, compliance, contracts, appraisals, imports |
| **Manager** | Team work items, leave approvals, reporting, audit |
| **Staff** | Own work items, leave requests, compliance self-service |
| **Read-Only** | View-only across all modules — no mutations |

## Self-signup is disabled

Accounts can only be created by an admin or HR Ops user. There is no public sign-up. Contact your administrator to get access.

## Getting help

Contact the DCS team or raise an issue in the ops channel.
