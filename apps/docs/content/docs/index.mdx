# DCS Ops Center

Welcome to the internal documentation for **DCS Ops Center** — the operations management platform for NDMA Data Centre Services (DCS).

## What is DCS Ops Center?

DCS Ops Center replaces spreadsheets and WhatsApp coordination for the day-to-day running of NDMA's data centre. It covers:

| Module | Purpose |
|--------|---------|
| **Work Management** | Task tracking, weekly updates, overdue monitoring |
| **Incident Management** | Incident response, timelines, post-incident reviews |
| **On-Call Roster** | Schedule management, swaps, on-call history |
| **Leave** | Leave requests, balances, team calendar |
| **Access & Accounts** | Platform accounts, VPN, identity governance |
| **Procurement** | Purchase requisitions, approvals, vendor tracking |
| **Compliance** | Training records, PPE logs, policy acknowledgements |
| **Audit** | Immutable audit trail for all system changes |

## Quick start

1. Sign in at the DCS Ops Center app
2. Your role determines what you can access and modify
3. All changes are logged in the Audit module

## Roles

| Role | Access level |
|------|-------------|
| **Admin** | Full access to all modules and settings |
| **HR / Admin Ops** | Staff, leave, compliance, contracts |
| **Manager** | Team work items, approvals, reporting |
| **Staff** | Own work items, leave requests, compliance |
| **Read-Only** | View-only access across all modules |
